pandora
=======

Installation from git
---------------------
1. Clone the repository:

		git clone git://github.com/mfrey/pandora.git

2. Install the dependencies for your distribution (packages names can differ):

        * Python
		  * pp 
		  * pygraphviz 
		  * networkx
		  * matplotlib
		* R
          * plyr
		  * lattice
		  * ggplot2
